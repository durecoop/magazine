# 두레생협 홍보 페이지 생성기

기획전 URL을 입력하면 백화점 카탈로그 스타일의 홍보 페이지를 자동 생성합니다.

## 사용법

```bash
npm install
npm start
```

http://localhost:3000 에서 실행됩니다.

## 기능

1. **URL 입력** — ecoop.or.kr 기획전 URL에서 상품 자동 파싱
2. **상품 확인** — 검색, 섹션 필터, AI 설명 생성, 선택/해제
3. **카테고리** — 기획전 상세 번호별 자동 분류, AI 테마명 추천, 순서 변경
4. **페이지 설정** — 글씨 크기/글씨체, 프로모션 소개, 이미지 첨부
5. **미리보기** — 데스크톱/모바일 미리보기, HTML 저장/복사

## AI 설명 생성 (선택)

Gemini API 키가 있으면 `.env` 파일에 추가:

```
GEMINI_API_KEY=your_key_here
```

키가 없어도 로컬 자동 생성으로 동작합니다.
