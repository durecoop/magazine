1

🤖클로드 코드가 뭔가요?

한마디로: AI가 대신 코딩해주는 도구

클로드 코드(Claude Code)는 앤트로픽(Anthropic)이 만든 AI 코딩 비서예요. 여러분이 "이런 프로그램 만들어줘", "이 오류 고쳐줘"라고 말로 지시하면, AI가 알아서 코드를 작성하고, 파일을 수정하고, 테스트까지 해줍니다.

💡쉽게 비유하면

네비게이션이 운전을 대신해주는 것처럼,
클로드 코드는 코딩을 대신해주는 AI 네비게이션이에요.

목적지(만들고 싶은 것)만 말하면, 길(코드)을 알아서 찾아갑니다.

할 수 있는 것들

🔨프로그램 만들기

"회원가입 페이지 만들어줘" → 코드 작성 + 파일 생성 + 테스트

🐛버그 수정

"로그인하면 빈 화면이 떠요" → 원인 분석 + 코드 수정

📝코드 관리

"변경사항 커밋해줘" → Git 자동 관리 + PR 생성

🧪테스트 작성

"이 기능 테스트 코드 만들어줘" → 테스트 작성 + 실행 + 수정

📚코드 이해

"이 프로젝트 구조 설명해줘" → 전체 분석 + 요약

⬇️설치하기

3분이면 끝!

Mac / Linux

터미널 curl -fsSL https://claude.ai/install.sh | bash

Windows (PowerShell)

PowerShell irm https://claude.ai/install.ps1 | iex

Homebrew (Mac)

터미널 brew install --cask claude-code

설치 없이 쓰기

🚀처음 시작하기

3

5

⌨️핵심 명령어 10개

이것만 알면 80%는 쓸 수 있어요

명령어하는 일
claude 대화 모드 시작
claude "할 일" 한 줄 명령으로 바로 작업
claude -c 이전 대화 이어서 하기
claude -r 지난 대화 목록에서 골라서 이어하기
claude commit 변경사항 Git 커밋
claude -p "질문" 답변만 받고 바로 종료
claude update 최신 버전으로 업데이트
/help 도움말 보기 (대화 중)
/clear 대화 초기화 (대화 중)
Shift+Tab 모드 전환 (일반→자동승인→플랜)
6

🎛️3가지 사용 모드

Shift+Tab으로 전환! 상황에 맞게 골라 쓰세요

✋일반 모드 (Normal)

파일 수정 전에 매번 "이거 바꿔도 돼?" 물어봅니다.
처음 쓸 때, 중요한 파일 다룰 때 안전하게!

⏩자동 승인 모드 (Auto-Accept)

파일 수정을 자동으로 허락합니다.
빠르게 작업할 때 유용. 단, 실수도 자동 적용되니 주의!

📋플랜 모드 (Plan)

분석만 하고 수정은 안 합니다.
"이 코드 어떻게 고치면 좋을까?" 같은 상담용. 읽기 전용이라 안전!

💡 추천 사용법
처음엔 일반 모드로 시작 → 익숙해지면 자동 승인 → 복잡한 작업 전엔 플랜 모드로 계획 세우기

⚡잘 쓰는 법 (공식 팁)

같은 도구도 어떻게 쓰느냐에 따라 결과가 달라져요

① 구체적으로 말하세요

❌ 이렇게 하지 마세요
"버그 고쳐줘"
✅ 이렇게 하세요
"로그인 후 빈 화면이 뜨는 버그 고쳐줘"

② 복잡한 건 단계별로

1. 사용자 프로필용 DB 테이블 만들어줘
2. 프로필 조회/수정 API 만들어줘
3. 프로필 페이지 UI 만들어줘

③ 먼저 분석, 그다음 수정

복잡한 작업은 플랜 모드로 먼저 분석하고, 계획 확인 후 일반 모드로 실행하세요.

④ CLAUDE.md로 규칙 저장

프로젝트 폴더에 CLAUDE.md 파일을 만들면, 클로드가 매 세션마다 자동으로 읽습니다.

# CLAUDE.md 예시
- TypeScript 사용
- 함수명은 camelCase
- 테스트 필수 작성
- 한국어 주석 사용

⑤ 서브에이전트 활용 (고급)

복잡한 작업은 여러 AI가 나눠서 동시에 처리할 수 있어요.

코드 리뷰어 서브에이전트로 인증 모듈 검토해줘

❓자주 묻는 질문

코딩을 전혀 모르는데 쓸 수 있나요?
네! 웹 브라우저(claude.ai/code)나 데스크톱 앱을 쓰면 됩니다. "블로그 만들어줘", "엑셀 데이터 정리하는 프로그램 만들어줘"처럼 말로 지시하면 돼요. 다만 결과물을 이해하고 활용하려면 기초 지식이 있으면 더 좋습니다.
내 코드가 유출되지는 않나요?
Claude Pro/Max 구독 사용 시, 대화 내용과 코드는 AI 학습에 사용되지 않습니다. API 사용 시에도 마찬가지예요. 다만 민감한 비밀번호나 API 키는 코드에 직접 넣지 않는 것이 좋습니다.
ChatGPT랑 뭐가 다른가요?
ChatGPT는 대화형 AI이고, 클로드 코드는 실제 파일을 읽고, 수정하고, 명령어를 실행할 수 있는 코딩 전용 도구예요. 프로젝트 전체를 이해하고 여러 파일을 동시에 작업할 수 있다는 게 가장 큰 차이입니다.
어떤 프로그래밍 언어를 지원하나요?
거의 모든 언어를 지원합니다. Python, JavaScript, TypeScript, Java, C++, Go, Rust, Swift, Kotlin, HTML/CSS, SQL 등. 특히 Python과 JavaScript/TypeScript에서 가장 강력합니다.
인터넷 없이도 쓸 수 있나요?
아니요, 인터넷 연결이 필요합니다. AI 모델이 클라우드에서 동작하기 때문이에요.
내 파일을 AI가 마음대로 바꾸나요?
일반 모드에서는 매번 허락을 구합니다. "이 파일을 이렇게 바꿀까요?"라고 물어보고, 승인해야 적용돼요. 자동 승인 모드를 켜지 않는 한 안전합니다.
